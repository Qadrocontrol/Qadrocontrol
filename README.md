**Квадро Контрол** — это мощное мобильное приложение на базе **Flutter** для управления квадрокоптерами через микроконтроллер **ESP32** и полётный контроллер (Pixhawk/Cube) по протоколу **MAVLink**.

Приложение обеспечивает полный цикл взаимодействия с БПЛА: от подключения и арминга до тонкой настройки PID-регуляторов, калибровки датчиков и визуализации телеметрии в реальном времени с помощью 3D-модели.

---

## ✨ Особенности

*   📡 **Связь в реальном времени:** Телеметрия (GPS, батарея, ориентация) через WiFi с минимальной задержкой.
*   🎮 **Полное управление:** Виртуальные джойстики (Mode 2), арминг/дизарминг, взлет/посадка.
*   ⚙️ **Глубокая настройка FC:** Редактирование параметров MAVLink, калибровка акселерометра/гироскопа/компаса.
*   🎨 **Динамические темы:** 5 цветовых пресетов с мгновенным применением и сохранением.
*   🔒 **Безопасность:** Лицензионное соглашение, проверка совместимости прошивки, защита от случайного запуска.
*   📱 **Адаптивный UI:** Современный дизайн в стиле Cyberpunk/Material 3.

## 🛠 Архитектура системы

Система состоит из трех основных компонентов:
1.  **Мобильное приложение (Android):** Интерфейс, логика управления, визуализация.
2.  **Шлюз (ESP32):** Преобразует команды WiFi в UART (MAVLink) и передает телеметрию обратно.
3.  **Полётный контроллер (FC):** Pixhawk/Cube, управляющий моторами и датчиками.
